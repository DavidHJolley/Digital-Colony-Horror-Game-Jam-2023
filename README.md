# Digital-Colony-Horror-Game-Jam-2023

## Unity version - 2021.3.1f1

## Setting up Unity

 - Set up Unity with Git: You need to set up Unity with Git. Open your Unity project and go to Edit > Project Settings > Editor. Under "Version Control," select "Visible Meta Files" and "Force Text." This will ensure that all the Unity files are in a text format that can be easily tracked by Git.

- After setting up your environment, connect Unity to GitHub: In Unity, go to Window > GitHub > Settings and enter your GitHub credentials. Then, click on the "Connect to GitHub" button and select your repository from the list.
